## Introduction
This game is created for my undergraduate dissertation project to improve and test the reaction speed of the player. Players will be slapping the Tricky Moggy that pop out randomly but beware of the Good Moggy.

## Gameplay
![image](https://user-images.githubusercontent.com/9387781/69363797-06a8cc80-0c89-11ea-92f5-97b0c0dc2e34.png)

1. Tap on the Yellow Cat (Triggy Moggy) when he pops up.
2. Do not Tap on the Grey Cat (Good Moggy).
3. Slap them by following pop up order will start the combo streak.
